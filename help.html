Welcome to the GExplore User Guide

Use this guide to help you learn GExplore's features and accelerate your research and analysis. GExplore provides comprehensive data on genes, mutations, proteins, and their expressions, making it easier for you to explore and analyze genetic information.


Start at the beginning, or visit each section individually to work your way through a project session. Each section of this guide provides detailed information and instructions on how to use GExplore's features effectively.

Last updated on Aug 1, 2024

GExplore Overview

GExplore is a tool for large-scale mining of data related to gene or protein function in C. elegans. The interface is simple, and response times are fast to encourage large-scale exploratory searches. GExplore is designed to offer detailed insights into genetic data, enabling researchers and analysts to delve deep into genetic information.


The database contains selected genome-wide data sets, including protein domain organization, gene expression, and phenotype data, which are important indicators for gene and protein function. Data sets were obtained from Wormbase and processed to integrate data and to simplify searches.


This site should be useful for experimental planning of genome-scale experiments and survey-type queries. With this interface, you should be able to get quick answers to questions such as:

  • How many small secreted proteins are there?
  • Which putative cell surface receptors are expressed in neurons?
  • How many kinases expressed in muscle cells have mutants available?

We are continuously improving GExplore and its features. If you have any questions or feedback, please do not hesitate to contact Dr. Harald Hutter.

August 2024 release (version 1.5)

GExplore version 1.5 is a major update including the following novel data sets and features:

  • All databases were updated to Wormbase release 292.
  • The domain organization for the proteomes of all nine core species in Wormbase was added to allow searches for proteins with certain domains in different nematode species.
  • Gene ontology data were added for all species.
  • The location of C. elegans mutations was mapped onto the domain organization of the proteins.
  • SAGE and RNAseq expression data were replaced by new stage-specific expression profiles from the modENCODE project (provided by the Waterston lab).
  • Best BLAST hits were replaced by orthologs as listed by Wormbase; a search option to identify C. elegans orthologs of human, mouse, Drosophila, or yeast genes was added.

Gene Database

The gene database contains selected datasets relevant to predict the putative function of genes/proteins. These include the domain organization of the protein, the general gene description, phenotype and expression data, gene interaction data, as well as gene ontology and disease associations. All data were downloaded from Wormbase. The database will be regularly updated. The current version is based on Wormbase release 292 (April 2024).


The search interface allows the user to quickly identify groups of genes sharing certain features, e.g. all kinases that are expressed in muscle cells. Alternatively, the user can start with a large list (100s or 1000s) of candidate genes and filter them for certain characteristics. The quick response time and flexible output options allow for quick survey-type queries with large groups of genes in the input and/or output.

Mutation Database

Mutation data were downloaded from Wormbase (release 292, April 2024). Only exonic alleles and alleles affecting splice junctions are included. SNPs from wildtype isolates were excluded. The curated list of mutations should be enriched for alleles affecting the function of the protein. The location of the mutations was mapped onto the domain organization of the protein for a visual display of the location and nature of the mutations. For deletions a color code indicates whether the deletion is in-frame (purple) or causes a frameshift (red). Mutations were mapped to all known splice variants of the gene, so that splice-variant specific allele can be easily identified. For deletions the location of the endpoints is indicated ('outside' the gene, in an 'intron' or in an 'exon'). Display options include the possibility to show the protein sequence with the affected amino acid(s) highlighted. The WS292 version of the database contains 227,110 alleles in 19,661 genes (28,240 splice variants) with 12,998 deletions, 183,547 missense and 10,014 nonsense allele. The database contains an additional 8,763 alleles without molecular information.

Protein Database

The protein database contains data from 13 Caenorhabditis species (C. becei, C. brenneri, C. briggsae, C. elegans, C. japonica, C. parvicauda, C. quiockensis, C. remanei, C. sulstoni, C. uteleia, C. tropicalis, C. waitukubuli, C. zanzibari) representing all branches of the Caenorhabditis phylogenetic tree. In addition, the database contains proteome data from 7 other namatode species (B. malayi, O. tipulae, O. volvulus, P. pacificus, P. redivivus, S. ratti, T. muris) that have high-quality data available. For comparison the database also contains the proteomes of the major "model organisms" (S. cerevisiae, D. melanogaster, D. rerio, M. musculus, H. sapiens).


Protein data were downloaded either from Wormbase (nematode species) or the 'reference proteome site (https://www.ebi.ac.uk/reference_proteomes/) (model organisms). SMART was used for protein domain prediction. To allow for comparisons across species the database also contains information about homologs in all species in the database.


The search interface allows the user to selected proteins with certain protein domains or certain domain arrangements for a selected species. The output can be limited to proteins with homologs in one or more of the other species. This can be used to quickly identify either nematode-specific genes or evolutionary conserved genes of a certain type (e.g. kinases).

Compare Tool

Compare provides an easy way to find unique and common genes within two sets of genes (two lists of terms of any kind). The input fields accept a comma-separated list of terms such as the "list of genes/proteins" found on the results pages of GExplore. The out consists of three lists: the common genes and the unique genes from each of the two input sets.

Last updated on Aug 1, 2024

Convert Tool

The convert input field accepts any combination of C. elegans gene identifiers: Wormbase gene ID (WBGene00003738), gene sequence name (F54F3.1), locus (nid-1), protein sequence name (F54F3.1a), wormpep ID (CE18731), Uniprot ID (Q93791). The output consists of a table with all the identifiers for all the genes in the input field. This can be used to standardize identifiers that come from different sources or to convert one set of identifiers into another one.
